Run this function if you want to upload your changes to git hub

git-save

if this doesn't work, you need to do these in powershell

$PROFILE

New-Item -ItemType Directory -Path (Split-Path $PROFILE) -Force

notepad $PROFILE

after notepad is created put this in it

function git-save {
    param(
        [string]$message = "Quick commit"
    )

    Write-Host "Staging all changes..."
    git add .

    Write-Host "Committing with message: $message"
    git commit -m "$message"

    Write-Host "Pushing to remote repository..."
    git push

    Write-Host "âœ… All done!"
}

Save and close the notepad, then try git-save again
